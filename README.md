## 🚀 Comment Lancer Localement

### Prérequis
- Navigateur web moderne (Chrome, Firefox, Safari, Edge)
- Aucune dépendance Node.js ni build tool requis

### Instructions

1. **Cloner le dépôt** :
   ```bash
   git clone https://github.com/WalidHALLOUCHE/portfolio_walid.git
   cd portfolio_walid
   ```

2. **Ouvrir en local** :
   - Ouvrir directement `index.html` dans le navigateur
   - OU utiliser un serveur local :
     ```bash
     python -m http.server 8000
     ```
   - Puis naviguer vers `http://localhost:8000`

3. **Vérifier** :
   - CSS chargé ✓
   - Images chargées ✓
   - PDF accessibles ✓
   - Boutons de téléchargement fonctionnels ✓
   - Liens GitHub/LinkedIn actifs ✓

- **Fond** : `#07111f` (bleu nuit)

- **Accent secondaire** : `#38bdf8` (cyan)
- **Texte** : `#f8fafc` (blanc cassé)

### Typographie
- **Corps** : Inter (400, 500, 600, 700, 800, 900)
- **Code** : JetBrains Mono (500, 700)
- Hiérarchie : H1 > H2 > H3 avec des tailles fluides

### Animations
- Reveal au scroll (opacity + translateY)
- Hover effects sur cards et liens
- Transitions fluides (0.2s - 0.7s)
- Pas de jank, optimisé pour perf

### Responsive
- **Desktop** : 1180px max-width, grilles 4/2 colonnes
- **Tablet** : 980px breakpoint, grilles 2 colonnes
- **Mobile** : 760px breakpoint, menu hamburger, grilles 1 colonne

## 📥 Boutons de Téléchargement

Tous les boutons utilisent l'attribut HTML `download` :

```html

<!-- Affichage dans navigateur -->
```

## 🛠️ Maintenance

### Ajouter un nouveau projet
1. Dupliquer une `<article class="project-card">` dans la section `#projets`
2. Remplir : titre, contexte, problème, stack, lien GitHub
3. Ajouter badges appropriés
4. Rafraîchir le navigateur

### Modifier les couleurs
1. Éditer les CSS variables dans `assets/css/styles.css` (`:root`)
2. Mettre à jour `--bg`, `--accent`, `--text`, etc.

### Optimiser les images
- Utiliser WebP si possible
- Compresser JPEG à 70-80% qualité
- Conserver les dimensions d'origine
